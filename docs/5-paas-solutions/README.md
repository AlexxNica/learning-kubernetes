# What is PaaS?

A PaaS is simply an IDE for managing clusters.

All PaaS have workflows similar to this tool stack:

- A text editor
- Git (Version Control)
- CI Pipeline
- GUI Monitoring System

PaaS intergrates all of these into one (web-based) GUI.

# PaaS Market

- Docker EE (Swarm mode only)
- OpenShift Origin
- Deis
- Rancher 2.0
- Tectonic
